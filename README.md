# 🏥 DocAppoint — Doctor Appointment Manager

A modern doctor appointment booking system built with Next.js and Express.js.

## 🌐 Live URL

## ✨ Key Features

- 🔐 Secure authentication with Firebase (email/password + Google OAuth)
- 🩺 Browse and search doctors by specialty and name
- 📅 Book, update, and cancel appointments with ease
- 👤 Personal dashboard to manage bookings and profile
- 🔑 JWT-based API security for all protected endpoints
